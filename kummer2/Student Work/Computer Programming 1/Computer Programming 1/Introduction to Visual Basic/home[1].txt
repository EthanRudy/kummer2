<P>You are free to download these files and use them &#145;as is&#146; or alter them in any way (copyright free). To use these files with Visual Basic 5.0 you should click on the appropriate file title, specify a file name in the dialog box that appears and then exit the web browser and unzip the file. If you do not have any decompression software then jump to <A HREF="http://www.winzip.com">www.winzip.com</A>.</P>

<H3>Visual Basic 5.0 Projects:</H3>
<UL>
<LI><P><B><A HREF="ftp://ftp.dcs.napier.ac.uk/pub/s.wilkinson/VB50/database.zip">Access 97 Databases</A></B> (22K)<BR>
Writing a Visual Basic front-end to an Access 97 database.</P>

<LI><P><B><A HREF="ftp://ftp.dcs.napier.ac.uk/pub/s.wilkinson/VB50/activex.zip">ActiveX</A></B> (18K)<BR>
Code used in the ActiveX example from week 17.</P>

<LI><P><B><A HREF="ftp://ftp.dcs.napier.ac.uk/pub/s.wilkinson/VB50/animate.zip">Animation</A></B> (58K)<BR>
How to incorporate graphics on a form and to use animation.</P>

<LI><P><B><A HREF="ftp://ftp.dcs.napier.ac.uk/pub/s.wilkinson/VB50/crystal_reports.zip">Crystal Report</A></B> (24K)&nbsp;&nbsp;<IMG SRC="graphics/new.gif"><BR>
Exactly the same example as &#145;Access 97 Databases&#146; above but with the addition of a Crystal Report which lists book details sorted by publisher.</P>

<LI><P><B><A HREF="ftp://ftp.dcs.napier.ac.uk/pub/s.wilkinson/VB50/graph.zip">Graphs</A></B> (3K)<BR>
An example which shows how to alter the values of various points on a graph and how to control the type of graph (e.g. bar chart, line graph, polar, etc.).</P>

<LI><P><B><A HREF="ftp://ftp.dcs.napier.ac.uk/pub/s.wilkinson/VB50/flexgrid.zip">FlexGrid</A></B> (4K)<BR>
An example which shows a spreadsheet-like computer inventory which allows products to be added or deleted.</P>

<LI><P><B><A HREF="ftp://ftp.dcs.napier.ac.uk/pub/s.wilkinson/VB50/mdi.zip">MDI Interface</A></B> (3K)<BR>
A basic example of how to create an MDI text editor. This example also includes basic clipboard functionality to copy and paste text.</P>

<LI><P><B><A HREF="ftp://ftp.dcs.napier.ac.uk/pub/s.wilkinson/VB50/mensa.zip">Mensa Problem</A></B> (2K)<BR>
The solution of a Mensa problem using the brute force of three nested loops.</P>

<LI><P><B><A HREF="ftp://ftp.dcs.napier.ac.uk/pub/s.wilkinson/VB50/sound.zip">MMControl</A></B> (259K)<BR>
An example demonstrating how wave and midi sounds can be played using the MMControl.</P>

<LI><P><B><A HREF="ftp://ftp.dcs.napier.ac.uk/pub/s.wilkinson/VB50/map.zip">Pocket Map of Europe</A></B> (117K)&nbsp;&nbsp;<IMG SRC="graphics/new.gif"><BR>
This example demonstrates how the <I>PictureClip</I> and <I>Picture</I> controls can be used together to scroll around a large image.</P>

<LI><P><B><A HREF="ftp://ftp.dcs.napier.ac.uk/pub/s.wilkinson/VB50/theatre.zip">Theatre Booking System</A></B> (18K)<BR>
An example which uses two separate option button groups (using frames) and a spin button linked to a text box.</P>

<LI><P><B><A HREF="ftp://ftp.dcs.napier.ac.uk/pub/s.wilkinson/VB50/tictactoe.zip">Tic Tac Toe</A></B> (3K)<BR>
This game is used to demonstrate drag & drop, control arrays and programmer-defined functons.</P>
</UL>

<H3>Other:</H3>
<UL>
<LI><P><B><A HREF="ftp://ftp.dcs.napier.ac.uk/pub/s.wilkinson/VB50/vb_site.zip">Visual Basic 5.0 Web Site</A></B> (868K)<BR>
For those students who would like to utilise information from this site then a ZIP file of it can be downloaded. This can be useful for saving expensive connection charges and also for accessing the site even with no Internet access. All you need is an HTML browser (e.g. <A HREF="http://home.netscape.com/download/index.html">Netscape Navigator</A> or <A HREF="http://www.microsoft.com/ie/default.asp">Internet Explorer</A>) running locally on your home computer and the above file dezipped. Please note that all HTML pages and graphics from the site are included but the sample projects listed above must be downloaded separately.</P>

<LI><P><B><A HREF="ftp://ftp.dcs.napier.ac.uk/pub/s.wilkinson/VB50/tictactoe_help.zip">Windows Help File</A></B> (3K)<BR>
The help file and RTF source code to complement the Tic Tac Toe example above.</P>

<LI><P><B><A HREF="ftp://ftp.dcs.napier.ac.uk/pub/s.wilkinson/VB50/icohlic3.zip">Icoholic 3</A></B> (788K)<BR>
A collection of 1,500 icons suitable for use in Windows and Visual Basic.</P>
</UL>
